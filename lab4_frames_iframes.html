① Floating Frame – float:left on iframe, text flows around it

A floating frame (float iframe) is an iframe positioned so that surrounding text or content wraps around it — similar to a floated image. The <iframe> tag is given a float:left CSS property so that adjacent text fills the remaining horizontal space.

This is very useful for embedding supplementary content (like a map, a widget, or a secondary document) alongside your main text without breaking the page flow.
