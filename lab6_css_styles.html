Lab 6 – CSS: Inline, Internal & External Stylesheets

Demonstrates all three methods of applying CSS to HTML elements

① INLINE CSS

What is Inline CSS?

Inline CSS is applied directly to an HTML element using the style attribute. It has the highest specificity and overrides both internal and external CSS for that element.

🔴 This paragraph uses INLINE CSS: color:white, background:#e74c3c, padding:12px

🟣 This text is styled inline: color:#8e44ad, font-size:18px, italic

🟠 This box uses inline border: 3px dashed #f39c12 

<p style="color: white; background: #e74c3c; padding: 12px;">
    Inline styled text
</p>
② INTERNAL CSS

What is Internal CSS?

Internal CSS is written inside a <style> block in the <head> section. It applies styles to the entire page.

This heading is styled using internal CSS class "internal-heading"

    table { border-collapse: collapse; }
    th { background: #2c3e50; color: white; }
</style>
③ EXTERNAL CSS

What is External CSS?

External CSS is written in a separate .css file and linked via <link rel="stylesheet" href="lab6_style.css"> in the head.
